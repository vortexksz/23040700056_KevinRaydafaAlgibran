Tugas praktikum ini bertujuan dalam mempelajari akuisisi jaringan, hashing file PCAP yang merupakan rekam jejak jaringan, pembuatan blockchain, dan validasi blockchain

Tools yang digunakan berupa:
-Wireshark sebagai alat untuk menangkap jejak jaringan pada perangkat
-Google collab sebagai alat untuk membuat hash, program blockchain serta validasi blockchain

cara menjalankan program
1. Download file zip pada folder evidence
2. kemudian esktrak file tersebut untuk mendapat file PCAP
3. Setelah itu, file yang sudah di download, upload pada file ipnyb yang sudah disediakan
4. jalankan satu persatu program pada google colab agar lebih nudah memahami
